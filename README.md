# ToDesk 产品介绍（中文）

ToDesk 是一款跨平台远程连接与远程运维工具，覆盖 Windows、macOS、Linux、Android、iOS 等系统，可用于远程办公、技术支持、设备管理与无人值守运维。

## 核心能力

- 高性能远程连接：低延迟、稳定连接，适配复杂办公与图形化场景。
- 多场景技术支持：支持快速协助排障、统一设备管理与操作追溯。
- 跨平台兼容：电脑与移动端均可接入，便于企业与个人协同。
- 安全机制：支持多重安全策略与权限管理，提升远程访问安全性。
